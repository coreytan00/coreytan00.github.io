Network packet sniffer

Frustrated that I had issues with my latency while playing League of Legends, I sought to figure out what was wrong with my network connection. I suspected that it may have been a bandwidth issue, so I coded a packet sniffer that captures all traffic going in and out. Of course I could simply use a program like WireShark, but why not code it instaed? The user can specify the running time of the sniffer and is provided data about the amount of data caught in Megabits and the bandwidth usage during that time. Additionally, all ipv4 packets caught are written into a text file in detail. Upon completion of the project, I determined that bandwitdth was unfortunately not the issue, and the secret perpetrator is still out there.

Implementation: Python - sockets, struct, bit-wise operations

Search Engine: Indexer

I developed a search engine from scratch that is capable of handling tens of thousands of documents or web pages. It returns accurate search results with a query response time under 200ms by utilizing page rank, word stemming, bookkeeping, weighted tf-idf and simhash. Documents were stored in an inverted index for ease of access.

Implementation: Python - flask

Mobile Application: Repunzel

Repunzel is an image recognition pun generator created during my very first hackathon, LA Hacks 2019.
